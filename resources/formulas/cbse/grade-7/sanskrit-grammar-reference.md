## 2. SHABDRUPANI (शब्दरूपाणि — Noun Declensions)

### The 8 Vibhaktis at a Glance

| # | विभक्ति | English Case | Question | Hindi | Preposition |
|---|---------|:---:|---------|:---:|:---:|
| 1 | प्रथमा | Nominative | Who? कः? | (none)/ने | (none) |
| 2 | द्वितीया | Accusative | Whom/What? कम्? | को | to, towards |
| 3 | तृतीया | Instrumental | By/With? केन? | से/द्वारा | by, with |
| 4 | चतुर्थी | Dative | For whom? कस्मै? | के लिए | for, to |
| 5 | पञ्चमी | Ablative | From? कस्मात्? | से (origin) | from |
| 6 | षष्ठी | Genitive | Whose? कस्य? | का/के/की | of, 's |
| 7 | सप्तमी | Locative | Where/When? कस्मिन्? | में/पर | in, on, at |
| 8 | सम्बोधन | Vocative | (address) | हे! | O!, Hey! |

### अकारान्त पुल्लिंग — राम (Masculine, 'a' ending)

```
              एकवचनम्     द्विवचनम्     बहुवचनम्
              (Singular)   (Dual)        (Plural)
───────────────────────────────────────────────────
प्रथमा        रामः         रामौ          रामाः
द्वितीया       रामम्        रामौ          रामान्
तृतीया        रामेण        रामाभ्याम्      रामैः
चतुर्थी        रामाय        रामाभ्याम्      रामेभ्यः
पञ्चमी        रामात्        रामाभ्याम्      रामेभ्यः
षष्ठी         रामस्य       रामयोः        रामाणाम्
सप्तमी        रामे         रामयोः        रामेषु
सम्बोधन       हे राम!       हे रामौ!       हे रामाः!
```
**Pattern words:** बालकः, देवः, नरः, छात्रः, शिक्षकः

### आकारान्त स्त्रीलिंग — लता (Feminine, 'aa' ending)

```
              एकवचनम्     द्विवचनम्     बहुवचनम्
───────────────────────────────────────────────────
प्रथमा        लता          लते           लताः
द्वितीया       लताम्        लते           लताः
तृतीया        लतया         लताभ्याम्      लताभिः
चतुर्थी        लतायै        लताभ्याम्      लताभ्यः
पञ्चमी        लतायाः       लताभ्याम्      लताभ्यः
षष्ठी         लतायाः       लतयोः        लतानाम्
सप्तमी        लतायाम्      लतयोः        लतासु
सम्बोधन       हे लते!       हे लते!       हे लताः!
```
**Pattern words:** बालिका, सीता, रमा, कन्या, विद्या

### अकारान्त नपुंसकलिंग — फल (Neuter, 'a' ending)

```
              एकवचनम्     द्विवचनम्     बहुवचनम्
───────────────────────────────────────────────────
प्रथमा        फलम्         फले           फलानि
द्वितीया       फलम्         फले           फलानि
तृतीया        फलेन         फलाभ्याम्      फलैः
चतुर्थी        फलाय         फलाभ्याम्      फलेभ्यः
पञ्चमी        फलात्         फलाभ्याम्      फलेभ्यः
षष्ठी         फलस्य        फलयोः        फलानाम्
सप्तमी        फले          फलयोः        फलेषु
सम्बोधन       हे फल!        हे फले!       हे फलानि!
```
**Pattern words:** पुस्तकम्, जलम्, वनम्, नगरम्, पत्रम्

**Key Observation:** In neuter, प्रथमा = द्वितीया (always identical!)

## 4. DHATURUPANI (धातुरूपाणि — Verb Conjugations)

### Person-Number Grid

```
              एकवचन     द्विवचन     बहुवचन
प्रथमपुरुष    He/She/It  They(2)     They(3+)
(3rd Person)
मध्यमपुरुष    You(1)     You(2)      You(3+)
(2nd Person)
उत्तमपुरुष    I          We(2)       We(3+)
(1st Person)
```

**Remember: Sanskrit person order is REVERSED from English!**

### पठ् (Read) — All 5 Lakaaras

**लट् (Present):**
```
पठति      पठतः      पठन्ति
```

```
अपठम्     अपठाव     अपठाम
```

## 5. KARAKA (कारकम् — Semantic Roles)

### Quick Reference Card

```
WHO did it?        →  कर्ता (Doer)       →  प्रथमा   →  रामः
Did WHAT to?       →  कर्म (Object)      →  द्वितीया  →  पुस्तकम्
WITH what tool?    →  करण (Instrument)   →  तृतीया   →  लेखन्या
FOR whom?          →  सम्प्रदान (Receiver) →  चतुर्थी   →  बालकाय
FROM where?        →  अपादान (Source)    →  पञ्चमी   →  ग्रामात्
(Possession)       →  सम्बन्ध (Relation)  →  षष्ठी    →  रामस्य
WHERE/WHEN?        →  अधिकरण (Location)  →  सप्तमी   →  गृहे
HEY!               →  सम्बोधन (Address)   →  सम्बोधन  →  हे राम!
```

## 6. SANDHI (सन्धिः — Sound Combinations)

### स्वर सन्धि (Vowel Sandhi)

| Type | Rule | Example |
|------|------|---------|
| **दीर्घ** | Same vowel + Same = Long | अ+अ=आ: देव+अरि = देवारि |
| | | इ+ई=ई: कवि+ईश = कवीश |
| | | उ+उ=ऊ: गुरु+उपदेश = गुरूपदेश |
| **गुण** | अ/आ + इ/ई = ए | देव+इन्द्र = देवेन्द्र |
| | अ/आ + उ/ऊ = ओ | सूर्य+उदय = सूर्योदय |
| | अ/आ + ऋ = अर् | देव+ऋषि = देवर्षि |
| **वृद्धि** | अ/आ + ए/ऐ = ऐ | एक+एक = एकैक |
| | अ/आ + ओ/औ = औ | वन+ओषधि = वनौषधि |
| **यण्** | इ/ई + other vowel | इति+आदि = इत्यादि (इ→य्) |
| | उ/ऊ + other vowel | सु+आगतम् = स्वागतम् (उ→व्) |
| **अयादि** | ए + vowel = अय् | ने+अनम् = नयनम् |
| | ओ + vowel = अव् | भो+अनम् = भवनम् |

### Memory Aid for Vowel Sandhi

```
SAME + SAME = LONG         (दीर्घ)     → अ+अ=आ
अ + MIDDLE vowel = MIDDLE  (गुण)      → अ+इ=ए, अ+उ=ओ
अ + COMPOUND vowel = COMPOUND (वृद्धि) → अ+ए=ऐ, अ+ओ=औ
OTHER vowel + vowel = SEMI-VOWEL (यण्)  → इ→य्, उ→व्, ऋ→र्
```

### व्यञ्जन सन्धि (Consonant Sandhi — Common Rules)

| Rule | Example |
|------|---------|
| त् + vowel → द् | तत् + एव = तदेव |
| त् + ज/झ → ज् | सत् + जनः = सज्जनः |
| त् + ग/घ → द् | उत् + घाटनम् = उद्घाटनम् |
| म् + consonant → ं | सम् + गमः = संगमः |

### विसर्ग सन्धि (Visarga Sandhi — Common Rules)

| Rule | Example |
|------|---------|
| ः + अ → ओ + ऽ | रामः + अस्ति = रामोऽस्ति |
| ः + voiced consonant → ओ | मनः + रथः = मनोरथः |

## 8. PRATYAYA (प्रत्ययाः — Suffixes)

### कृत् प्रत्यय (Added to verb roots)

| प्रत्यय | English Equiv. | Meaning | Example |
|---------|:---:|---------|---------|
| **क्त** | -ed (past part.) | done | पठ् + क्त = पठितः (read/studied) |
| **क्त्वा** | having -ed | after doing | पठ् + क्त्वा = पठित्वा (having read) |
| **तुमुन्** | to (infinitive) | in order to | पठ् + तुमुन् = पठितुम् (to read) |
| **शतृ** | -ing (pres. part.) | doing | पठ् + शतृ = पठन् (reading/reader) |
| **शानच्** | -ing (middle) | doing (self) | सेव् + शानच् = सेवमानः (serving) |

### तद्धित प्रत्यय (Added to nouns)

| प्रत्यय | English Equiv. | Meaning | Example |
|---------|:---:|---------|---------|
| **मतुप्** | -ful, having | possessing | बुद्धि + मतुप् = बुद्धिमान् (intelligent) |
| **त्व** | -ness, -hood | quality (neuter) | मनुष्य + त्व = मनुष्यत्वम् (humanity) |
| **ता** | -ness | quality (fem.) | सुन्दर + ता = सुन्दरता (beauty) |

## 9. UPASARGA (उपसर्गाः — Prefixes)

### The 22 Sanskrit Prefixes

| उपसर्ग | English | Effect | Example |
|--------|---------|--------|---------|
| **प्र** | pro-, forth | forward | प्र + गम् = प्रगम् (proceed) |
| **परा** | away | defeat/reverse | परा + जय = पराजय (defeat) |
| **अप** | away, down | removal | अप + मान = अपमान (insult) |
| **सम्** | con-, together | union | सम् + गम = संगम (meeting) |
| **अनु** | after, along | following | अनु + वाद = अनुवाद (translation) |

## 11. SENTENCE STRUCTURE

### Word Order Comparison

```
English (SVO):  Ram     reads    a book.
                Subject  Verb     Object

Sanskrit (SOV): रामः    पुस्तकं   पठति।
                Subject  Object   Verb

Hindi (SOV):    राम     किताब    पढ़ता है।
                Subject  Object   Verb
```

**Key Insight:** Sanskrit word order is flexible because vibhakti endings encode the grammatical role. All these mean "Ram reads a book":
- रामः पुस्तकं पठति। (standard SOV)
- पुस्तकं रामः पठति। (OSV)
- पठति रामः पुस्तकं। (VSO)

## 12. COMMON ERRORS TO AVOID

| Error | Correction | Rule |
|-------|-----------|------|
| बालकाः गच्छति | बालकाः गच्छन्ति | Plural subject → plural verb |
| रामः पठन्ति | रामः पठति | Singular subject → singular verb |
| सीता गच्छति → सीतः | सीता correct | सीता is feminine, not masculine |
| "अ" in लङ् is separate word | Inseparable prefix | अपठत् not अ पठत् |
| बालिका + आलयः = बालिकाआलयः | बालिकालयः | दीर्घ सन्धि: आ+आ=आ |
