## 2. SHABDRUPANI (शब्दरूपाणि — Noun Declensions)

### The 8 Vibhaktis at a Glance

| # | विभक्ति | English Case | Question | Hindi | Preposition |
|---|---------|:---:|---------|:---:|:---:|
| 1 | प्रथमा | Nominative | Who? कः? | (none)/ने | (none) |
| 2 | द्वितीया | Accusative | Whom/What? कम्? | को | to, towards |
| 3 | तृतीया | Instrumental | By/With? केन? | से/द्वारा | by, with |
| 4 | चतुर्थी | Dative | For whom? कस्मै? | के लिए | for, to |
| 5 | पञ्चमी | Ablative | From? कस्मात्? | से (origin) | from |
| 6 | षष्ठी | Genitive | Whose? कस्य? | का/के/की | of, 's |
| 7 | सप्तमी | Locative | Where/When? कस्मिन्? | में/पर | in, on, at |
| 8 | सम्बोधन | Vocative | (address) | हे! | O!, Hey! |

### अकारान्त पुल्लिंग — राम (Masculine, 'a' ending)

```
              एकवचनम्     द्विवचनम्     बहुवचनम्
              (Singular)   (Dual)        (Plural)
───────────────────────────────────────────────────
प्रथमा        रामः         रामौ          रामाः
द्वितीया       रामम्        रामौ          रामान्
तृतीया        रामेण        रामाभ्याम्      रामैः
चतुर्थी        रामाय        रामाभ्याम्      रामेभ्यः
पञ्चमी        रामात्        रामाभ्याम्      रामेभ्यः
षष्ठी         रामस्य       रामयोः        रामाणाम्
सप्तमी        रामे         रामयोः        रामेषु
सम्बोधन       हे राम!       हे रामौ!       हे रामाः!
```
**Pattern words:** बालकः, देवः, नरः, छात्रः, शिक्षकः

```
───────────────────────────────────────────────────
```

```
───────────────────────────────────────────────────
प्रथमा        फलम्         फले           फलानि
द्वितीया       फलम्         फले           फलानि
तृतीया        फलेन         फलाभ्याम्      फलैः
```

## 4. DHATURUPANI (धातुरूपाणि — Verb Conjugations)

### Person-Number Grid

```
              एकवचन     द्विवचन     बहुवचन
प्रथमपुरुष    He/She/It  They(2)     They(3+)
(3rd Person)
मध्यमपुरुष    You(1)     You(2)      You(3+)
(2nd Person)
उत्तमपुरुष    I          We(2)       We(3+)

```

**Remember: Sanskrit person order is REVERSED from English!**

### पठ् (Read) — All 5 Lakaaras

**लट् (Present):**
```
पठति      पठतः      पठन्ति
पठसि      पठथः      पठथ
पठामि     पठावः     पठामः
```

**लङ् (Past):**
```
अपठत्     अपठताम्    अपठन्
अपठः      अपठतम्    अपठत
अपठम्     अपठाव     अपठाम
```

**लृट् (Future):**

> **Note:** NCERT textbooks often label this as लुट्. Technically these forms are लृट् (first future). Both names refer to the same conjugation at school level.

```
पठिष्यति   पठिष्यतः   पठिष्यन्ति
पठिष्यसि   पठिष्यथः   पठिष्यथ
पठिष्यामि  पठिष्यावः  पठिष्यामः
```

**लोट् (Imperative/Command):**
```
पठतु      पठताम्     पठन्तु
पठ        पठतम्     पठत
पठानि     पठाव      पठाम
```

**विधिलिङ् (Should/Potential):**
```
पठेत्      पठेताम्     पठेयुः
पठेः       पठेतम्     पठेत
पठेयम्     पठेव      पठेम
```

### आत्मनेपद (Middle Voice) — Verbs "for oneself"

**Difference:** परस्मैपद verbs (पठति) = action for others. आत्मनेपद verbs (सेवते) = action benefits the doer.

**Quick identification:** परस्मैपद ends in -ति/-न्ति, आत्मनेपद ends in -ते/-न्ते

#### आत्मनेपद vs परस्मैपद Endings (लट्)

```
              परस्मैपद          आत्मनेपद
              (For others)       (For self)
───────────────────────────────────────────────
प्रथमपुरुष   ति   तः   न्ति     ते   एते   अन्ते
मध्यमपुरुष   सि   थः   थ      से   एथे   ध्वे
उत्तमपुरुष   मि   वः   मः     ए    आवहे  आमहे
```

## 6. SANDHI (सन्धिः — Sound Combinations)

### स्वर सन्धि (Vowel Sandhi)

| Type | Rule | Example |
|------|------|---------|
| **दीर्घ** | Same vowel + Same = Long | अ+अ=आ: देव+अरि = देवारि |
| | | इ+ई=ई: कवि+ईश = कवीश |
| | | उ+उ=ऊ: गुरु+उपदेश = गुरूपदेश |
| **गुण** | अ/आ + इ/ई = ए | देव+इन्द्र = देवेन्द्र |
| | अ/आ + उ/ऊ = ओ | सूर्य+उदय = सूर्योदय |
| | अ/आ + ऋ = अर् | देव+ऋषि = देवर्षि |
| **वृद्धि** | अ/आ + ए/ऐ = ऐ | एक+एक = एकैक |
| | अ/आ + ओ/औ = औ | वन+ओषधि = वनौषधि |
| **यण्** | इ/ई + other vowel | इति+आदि = इत्यादि (इ→य्) |
| | उ/ऊ + other vowel | सु+आगतम् = स्वागतम् (उ→व्) |
| **अयादि** | ए + vowel = अय् | ने+अनम् = नयनम् |
| | ओ + vowel = अव् | भो+अनम् = भवनम् |

### Memory Aid for Vowel Sandhi

```
SAME + SAME = LONG         (दीर्घ)     → अ+अ=आ
अ + MIDDLE vowel = MIDDLE  (गुण)      → अ+इ=ए, अ+उ=ओ
अ + COMPOUND vowel = COMPOUND (वृद्धि) → अ+ए=ऐ, अ+ओ=औ
OTHER vowel + vowel = SEMI-VOWEL (यण्)  → इ→य्, उ→व्, ऋ→र्
```

### व्यञ्जन सन्धि (Consonant Sandhi — Rules)

| Rule Name | Rule | Example |
|-----------|------|---------|
| **जश्त्व** | 1st/2nd letter of वर्ग before voiced → 3rd letter | वाक् + ईशः = वागीशः (क्→ग्) |
| | | दिक् + गजः = दिग्गजः (क्→ग्) |
| **श्चुत्व** | त/थ/द/ध/न + श/च/छ/ज/झ/ञ → palatal | उत् + चारणम् = उच्चारणम् (त्→च्) |
| | | सत् + चित् = सच्चित् (त्→च्) |
| **ष्टुत्व** | त/थ/द/ध/न + ष/ट/ठ/ड/ढ/ण → retroflex | तत् + टीका = तट्टीका (त्→ट्) |
| **छत्वम्** | vowel + छ → vowel + च्छ (optional) | अनु + छेदः = अनुच्छेदः |
| **अनुनासिक** | 1st/2nd of वर्ग before nasal → 5th of same group | वाक् + मयम् = वाङ्मयम् (क्→ङ्) |
| | | जगत् + नाथः = जगन्नाथः (त्→न्) |
| **म् + consonant** | म् → anusvara (ं) before consonant | सम् + गमः = संगमः |
| **त् + vowel** | त् → द् before vowel | तत् + एव = तदेव |
| **त् + श** | → च्छ | तत् + शिवः = तच्छिवः |
| **त् + ह** | → द्ध | तत् + हितम् = तद्धितम् |

### विसर्ग सन्धि (Visarga Sandhi — Common Rules)

| Rule | Example |
|------|---------|
| ः + अ → ओ + ऽ | रामः + अस्ति = रामोऽस्ति |
| ः + voiced consonant → ओ | मनः + रथः = मनोरथः |

## 8. PRATYAYA (प्रत्ययाः — Suffixes)

### कृत् प्रत्यय (Added to verb roots)

| प्रत्यय | English Equiv. | Meaning | Example |

| **क्त** | -ed (past part.) | done | पठ् + क्त = पठितः (read/studied) |

#### Gender Forms (तव्यत् & अनीयर् change with gender!)

```
तव्यत् → पठितव्यः (पु.), पठितव्या (स्त्री.), पठितव्यम् (नपु.)
अनीयर् → पठनीयः (पु.), पठनीया (स्त्री.), पठनीयम् (नपु.)
```

> **Important (from Muktavali):** तव्यत् expresses विधिलिङ् लकार (potential mood) meaning — "should/must". Used mostly in कर्मवाच्य (passive voice). अनीयर् is used for आशीर्वाद (blessings), उपदेश (advice), and संभावना (possibility).

#### तव्यत् vs अनीयर् — When to use which?

| | तव्यत् (Must-do) | अनीयर् (Worth-doing) |
|--|--|--|
| **Meaning** | Obligation, duty (विधिलिङ् अर्थ) | Advice, blessing, possibility |
| **Usage** | कर्मवाच्य (passive voice) | कर्मवाच्य / भाववाच्य |
| **English** | must be, should be | worthy of, -able/-ible |
| **Example** | कर्तव्यम् = must be done (duty) | करणीयम् = worth doing (advice) |
| **Tone** | Stronger (command/obligation) | Softer (blessing/suggestion) |

## 11. SENTENCE STRUCTURE

### Word Order Comparison

```
English (SVO):  Ram     reads    a book.
                Subject  Verb     Object

Sanskrit (SOV): रामः    पुस्तकं   पठति।
                Subject  Object   Verb

Hindi (SOV):    राम     किताब    पढ़ता है।
                Subject  Object   Verb
```

**Key Insight:** Sanskrit word order is flexible because vibhakti endings encode the grammatical role. All these mean "Ram reads a book":
- रामः पुस्तकं पठति। (standard SOV)
- पुस्तकं रामः पठति। (OSV)
- पठति रामः पुस्तकं। (VSO)

## 15. SAMAASA (समासः — Compound Words)

### 4 Main Types of Samaasa

| Type | Meaning | Rule | Example |
|------|---------|------|---------|
| **तत्पुरुष** | Determinative | विभक्ति-based, second word is principal | राजपुत्रः = राज्ञः पुत्रः (prince — son of a king) |
| **द्वन्द्व** | Copulative (and) | Both words are equal | रामलक्ष्मणौ = रामः च लक्ष्मणः च (Rama and Lakshmana) |
| **बहुव्रीहि** | Possessive | Meaning refers to a third entity | पीताम्बरः = पीतम् अम्बरं यस्य सः (one whose clothes are yellow = Vishnu) |
| **अव्ययीभाव** | Adverbial | First word is an अव्यय, compound is also अव्यय | प्रतिदिनम् = दिने दिने (every day) |

### More Examples

- **तत्पुरुष:** राजगृहम् = राज्ञः गृहम् (king's house)
- **द्वन्द्व:** माता-पितरौ = माता च पिता च (mother and father)
- **बहुव्रीहि:** चक्रपाणिः = चक्रं पाणौ यस्य सः (one who has a discus in hand = Vishnu)
- **अव्ययीभाव:** उपकूलम् = कूलस्य समीपे (near the bank)
