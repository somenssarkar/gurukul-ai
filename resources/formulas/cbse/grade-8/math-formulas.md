## Chapter 1: Rational Numbers

| Property | Formula |
|----------|---------|
| Closure (Add/Mul) | a/b ± c/d and a/b × c/d are always rational |
| Commutative (Add) | a/b + c/d = c/d + a/b |
| Commutative (Mul) | a/b × c/d = c/d × a/b |
| Associative (Add) | (a/b + c/d) + e/f = a/b + (c/d + e/f) |
| Additive Identity | a/b + 0 = a/b |
| Multiplicative Identity | a/b × 1 = a/b |
| Additive Inverse | a/b + (−a/b) = 0 |
| Multiplicative Inverse | a/b × b/a = 1 (a, b ≠ 0) |
| Distributive | a/b × (c/d + e/f) = a/b × c/d + a/b × e/f |

**Operations:**
- **Add/Subtract:** a/b ± c/d = (ad ± bc) / bd
- **Multiply:** a/b × c/d = ac / bd
- **Divide:** a/b ÷ c/d = ad / bc
- **Standard form:** denominator positive, HCF(|p|, q) = 1

## Chapter 2: Exponents

| Law | Formula |
|-----|---------|
| Product | aᵐ × aⁿ = aᵐ⁺ⁿ |
| Quotient | aᵐ ÷ aⁿ = aᵐ⁻ⁿ (a ≠ 0) |
| Power of power | (aᵐ)ⁿ = aᵐⁿ |
| Power of product | (ab)ⁿ = aⁿ × bⁿ |
| Power of quotient | (a/b)ⁿ = aⁿ/bⁿ |
| Zero exponent | a⁰ = 1 (a ≠ 0) |
| Negative exponent | a⁻ⁿ = 1/aⁿ |

- **Standard form:** a × 10ⁿ where 1 ≤ a < 10
- Large number: positive n (e.g., 6.02 × 10²³)
- Small number: negative n (e.g., 4.5 × 10⁻⁵)

## Chapter 3: Squares and Square Roots

**Key Identities:**

| Identity | Formula |
|----------|---------|
| Perfect square | (a + b)² = a² + 2ab + b² |
| Perfect square | (a − b)² = a² − 2ab + b² |
| Difference of squares | (a + b)(a − b) = a² − b² |

- **Sum of first n odd numbers = n²** (e.g., 1+3+5+7 = 4² = 16)
- **Pythagorean triplet:** (2m, m²−1, m²+1) for any m > 1
- **√(a × b) = √a × √b**
- **√(a/b) = √a / √b**

**Square Root Methods:**
1. **Prime Factorisation:** group prime factors in pairs
2. **Long Division:** group digits in pairs from right; iterative division

## Chapter 5: Playing with Numbers

- **2-digit number (tens = a, units = b):** 10a + b
- **Reversed 2-digit number:** 10b + a; Difference = 9|a−b|; Sum = 11(a+b)
- **3-digit number (hundreds=a, tens=b, units=c):** 100a + 10b + c

**Divisibility Rules:**

| Divisible by | Rule |
|-------------|------|
| 2 | Last digit is 0, 2, 4, 6, or 8 |
| 3 | Sum of digits divisible by 3 |
| 4 | Last two digits divisible by 4 |
| 5 | Last digit is 0 or 5 |
| 6 | Divisible by both 2 and 3 |
| 8 | Last three digits divisible by 8 |
| 9 | Sum of digits divisible by 9 |
| 10 | Last digit is 0 |
| 11 | (Sum of odd-position digits) − (sum of even-position digits) = 0 or multiple of 11 |

**Standard Identities:**

| Identity | Formula |

| Square of sum | (a + b)² = a² + 2ab + b² |
| Square of difference | (a − b)² = a² − 2ab + b² |

- **Adding like terms:** axⁿ + bxⁿ = (a+b)xⁿ
- **Multiplying variables:** xᵐ × xⁿ = xᵐ⁺ⁿ
- **Division algorithm:** Dividend = Divisor × Quotient + Remainder

## Chapter 7: Factorisation

**Factorisation Methods:**

| Method | When to use | Formula |
|--------|------------|---------|
| Common factor | All terms share HCF | ab + ac = a(b + c) |
| Difference of squares | a² − b² form | (a + b)(a − b) |
| Perfect square trinomial | a² ± 2ab + b² | (a ± b)² |
| Sum of cubes | a³ + b³ | (a + b)(a² − ab + b²) |
| Difference of cubes | a³ − b³ | (a − b)(a² + ab + b²) |
| Trinomial | x² + bx + c | (x + p)(x + q) where p+q=b, pq=c |
| Regrouping | 4 terms with 2+2 pattern | Group and factor pairs |

## Chapter 8: Linear Equations

- **Standard form:** ax + b = 0 → x = −b/a
- **Transposition:** moving a term reverses its operation (+ ↔ −, × ↔ ÷)
- **Balance principle:** same operation on both sides
- **Cross multiplication:** a/b = c/d → ad = bc
- **Always verify:** substitute solution back into original equation

**Useful equation setups:**

| Problem Type | Setup |
|-------------|-------|
| Two consecutive integers | n, n+1 |
| Two consecutive even/odd integers | n, n+2 |
| Age problem (x years from now) | current age + x |
| Age problem (x years ago) | current age − x |

## Chapter 9: Percentage

| Conversion | Formula |
|------------|---------|
| Fraction → % | Multiply by 100 |
| % → Fraction | Divide by 100 |
| Decimal → % | Multiply by 100 |
| % → Decimal | Divide by 100 |

- **x% of N** = (x/100) × N
- **A as % of B** = (A/B) × 100
- **% increase** = (increase/original) × 100
- **% decrease** = (decrease/original) × 100
- **New value after x% increase** = original × (100 + x)/100
- **New value after x% decrease** = original × (100 − x)/100
- **Discount** = Marked Price − Selling Price
- **Discount%** = (Discount/Marked Price) × 100
- **SP after discount** = MP × (100 − Discount%)/100

## Chapter 10: Profit and Loss

| Find | Formula |
|------|---------|
| Profit | SP − CP |
| Loss | CP − SP |
| Profit % | (Profit/CP) × 100 |
| Loss % | (Loss/CP) × 100 |
| SP (given profit %) | CP × (100 + Profit%)/100 |
| SP (given loss %) | CP × (100 − Loss%)/100 |
| CP (given SP & profit %) | SP × 100/(100 + Profit%) |
| CP (given SP & loss %) | SP × 100/(100 − Loss%) |

- **Net SP after two successive discounts d₁ and d₂:**
  SP = MP × (100−d₁)/100 × (100−d₂)/100
- **Effective single discount** = d₁ + d₂ − (d₁×d₂)/100

## Chapter 11: Compound Interest

| Find | Formula |
|------|---------|
| Simple Interest | SI = PRT/100 |
| Amount (SI) | A = P + SI = P(1 + RT/100) |
| Amount (CI, annual) | A = P(1 + R/100)ⁿ |
| Compound Interest | CI = A − P |
| Amount (half-yearly) | A = P(1 + R/200)²ⁿ |
| Amount (quarterly) | A = P(1 + R/400)⁴ⁿ |

- **Difference CI − SI for 2 years:** CI − SI = P(R/100)²
- **Depreciation / Growth:** V = P(1 ± r/100)ⁿ

## Chapter 14: Polygons

| Shape | Sides | Sum of Interior Angles |
|-------|-------|------------------------|
| Triangle | 3 | 180° |
| Quadrilateral | 4 | 360° |
| Pentagon | 5 | 540° |
| Hexagon | 6 | 720° |
| Heptagon | 7 | 900° |
| Octagon | 8 | 1080° |
| Decagon | 10 | 1440° |

- **Sum of interior angles of n-gon** = (n − 2) × 180°
- **Each interior angle of regular n-gon** = (n − 2) × 180° / n
- **Each exterior angle of regular n-gon** = 360° / n
- **Interior + Exterior = 180°**
- **Sum of all exterior angles** = 360° (any convex polygon)

## Chapter 15: Quadrilaterals

- **Sum of all angles** = 360°

| Shape | Key Properties |
|-------|---------------|
| Trapezium | One pair of parallel sides |
| Parallelogram | Both pairs of opposite sides parallel and equal |
| Rhombus | All sides equal; diagonals bisect at 90° |
| Rectangle | All angles 90°; diagonals equal |
| Square | All sides equal; all angles 90°; diagonals equal and bisect at 90° |
| Kite | Two pairs of adjacent equal sides; one diagonal bisects the other |

## Chapter 17: Construction of Quadrilaterals

- **5 measurements needed** to uniquely determine a quadrilateral
- **Possible sets of 5:** (4 sides + 1 diagonal), (3 sides + 2 diagonals), (2 diagonals + 3 sides), (4 sides + 1 included angle), (3 sides + 2 included angles)
- **Steps:** draw base → construct triangles using diagonals/angles → complete quadrilateral

## Chapter 20: Volume and Surface Area of Solids

### Cuboid and Cube

| Measure | Cuboid | Cube |
|---------|--------|------|
| Total Surface Area (TSA) | 2(lb + bh + hl) | 6a² |
| Lateral Surface Area (LSA) | 2h(l + b) | 4a² |
| Volume | l × b × h | a³ |
| Diagonal | √(l² + b² + h²) | a√3 |

### Cylinder

| Measure | Formula |
|---------|---------|
| Curved Surface Area (CSA) | 2πrh |
| Total Surface Area (TSA) | 2πr(r + h) |
| Volume | πr²h |

### Cone

| Measure | Formula |
|---------|---------|
| Slant height | l = √(r² + h²) |
| Curved Surface Area (CSA) | πrl |
| Total Surface Area (TSA) | πr(r + l) |
| Volume | (1/3)πr²h |

### Sphere and Hemisphere

| Measure | Sphere | Hemisphere |
|---------|--------|-----------|
| Curved Surface Area | 4πr² | 2πr² |
| Total Surface Area | 4πr² | 3πr² |
| Volume | (4/3)πr³ | (2/3)πr³ |

- **1 litre = 1000 cm³ = 1 dm³**

## Chapter 21: Data Handling

| Measure | Formula |
|---------|---------|
| Mean (direct) | x̄ = Σfx / Σf |
| Mean (assumed mean A) | x̄ = A + Σfd/Σf (d = x − A) |
| Class mark (mid-value) | (Lower boundary + Upper boundary) / 2 |
| Class width | Upper boundary − Lower boundary |

- **Median class:** class where cumulative frequency first reaches or exceeds n/2
- **Modal class:** class with the highest frequency
- **Histogram:** continuous bars (no gaps), bar width = class width

## Chapter 23: Line Graphs and Linear Graphs

- **Linear equation:** y = mx + c
  - m = slope (gradient) = rise/run = Δy/Δx
  - c = y-intercept (value of y when x = 0)
- **Graph of y = kx:** straight line through origin (direct proportion)
- **Horizontal line:** y = constant (zero slope)
- **Vertical line:** x = constant (undefined slope)

## Chapter 24: Pie Charts

- **Central angle** = (Category value / Total value) × 360°
- **Percentage of category** = (Category value / Total value) × 100
- **Value from angle** = (Angle / 360°) × Total value
- **Sum of all central angles** = 360°

## Chapter 25: Probability

| Formula | Expression |
|---------|------------|
| Probability of event E | P(E) = Favourable outcomes / Total outcomes |
| Certain event | P(E) = 1 |
| Impossible event | P(E) = 0 |
| Complementary event | P(not E) = 1 − P(E) |
| Range | 0 ≤ P(E) ≤ 1 |

**Common sample spaces:**
- **Coin:** {H, T} → 2 outcomes
- **Two coins:** {HH, HT, TH, TT} → 4 outcomes
- **Die:** {1, 2, 3, 4, 5, 6} → 6 outcomes
- **Two dice:** 6 × 6 = 36 outcomes

**Useful die probabilities:**

| Event | Favourable | P(E) |
|-------|-----------|------|
| Even number | {2, 4, 6} | 1/2 |
| Odd number | {1, 3, 5} | 1/2 |
| Prime number | {2, 3, 5} | 1/2 |
| Multiple of 3 | {3, 6} | 1/3 |
| Number > 4 | {5, 6} | 1/3 |
